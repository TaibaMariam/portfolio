# Tayyaba Mariam — Portfolio (GitHub Pages)

## Deploy on GitHub Pages
1. Create a new GitHub repository (e.g. `tayyaba-portfolio`).
2. Upload the contents of this folder (index.html + assets).
3. In GitHub: Settings → Pages → Source: Deploy from a branch → Branch: `main` → Folder: `/ (root)`.
4. Your site will be live at: https://<username>.github.io/<repo>/

## Update images
Replace or add images in `assets/images/` and refresh the page.
